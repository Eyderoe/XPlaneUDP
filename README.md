XPlane UDP
===============================
XPlane12 & MinGW32 10.3 下通过测试

### 备注

- 发送格式：消息类型char[4] + 空字符char[0] + 数据结构
- 支持数据类型：CHR 字符、INT 4字节整数、FLT 4字节浮点、DOB 4字节双精度
